## [1.2.0] — 2026-03-06

### Added
- `manifest.json` — Web App Manifest enabling PWA installation
- `robots.txt` — search engine crawling directives
- `sitemap.xml` — XML sitemap for improved SEO
- `404.html` — custom Windows 11-styled 404 error page
- `LICENSE` — MIT license

## [1.1.0] — 2026-03-06

### Added
- `favicon.svg` — SVG favicon with Windows logo
- `.gitignore` — standard ignores for OS and editor files
- Lock screen live clock

### Fixed
- Lock screen time display on initial load

## [1.0.0] — 2026-03-06

### Added
- Initial release of the Windows 11-inspired interactive portfolio
- Realistic lock screen with live clock
- Fully draggable, minimizable, and maximizable application windows
- Interactive taskbar with Start Menu, Weather widget, Control Center, and Calendar
- Desktop icons and custom right-click context menu
- Desktop drag-to-select functionality
- Integrated apps: This PC, About Me, Projects, Contact, GitHub, LinkedIn, Instagram, Facebook, YouTube
- Dynamic weather and geolocation via `ipapi.co` and `api.open-meteo.com`
